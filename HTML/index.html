This is an example of playing and controlling audio.This is one of the features wich differentiates html4 and 5, this feature helps us play and control audio within our page. use the audiotag and specify the source of the mp3 or any audio file and then include 'controls' next to the opening audio tag, to enable sound controls and play and pause controls. this is very important.

This is an example of playing and controlling video.This is another feature wich differentiates html4 and 5, this feature helps us play and control video within our page. use the videotag and specify the source of the mp4 file or any video file and then include 'controls' next to the opening video tag, to enable sound controls and play and pause controls. this is very important.we can add an optional attribute 'autoplay' similar to the 'controls' attribute we added for audio.
