API Testing using Rest Assured (BDD Approach)

Prerequisites
Java
Eclipse
Maven
TestNg
Rest Assured Dependencies

Rest Assured Dependencies
rest-assured
json-schema-validator
json-path
xml-path
testng
java-hamcrest
hamcrest-core
hamcrest-library

The last 3 dependencies are required for BDD approach. Instead of downloading them seperately, we can download hamcrest-all dependency alone.

Keywords used in BDD
given - set cookies info, authorization info, parameters, headers info- basically prerequisites
when - to send get,put,delete requests etc. - basically conditions
then - validating status code, extract headers, body, cookies etc. - basically response validation

Validations:
Status code
status line
response
headers

Demo website:
GET Request
restapi.demoqa.com/utilities/weather/city/Chennai

imports are very important
import static io.restassured.RestAssured.*;
import static org.hamcrest.Matchers.*;
rest assured and hamcrest packages are not normal imports, they are static imports
static packages appear in different colors under maven dependencies section (dark grey)
