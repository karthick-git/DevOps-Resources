to perform a GET request in cmder
url = 'http://jsonplaceholder.typicode.com/photos'
response=requests.get(url)
response.json()

to perform a POST request in cmder
jsonPayload = {'albumId':1,'title':'test','url':'nothing.com','thumbnailUrl':'nothing.com'}
response=requests.post(url,json=jsonPayload)
response.json()

to perform a PUT request in cmder
url = 'http://jsonplaceholder.typicode.com/photos/100'
response=requests.put(url,json=jsonPayload)
response.json()

For websites with authentication
url = 'https://api.github.com/user'
response=requests.get(url,auth=('username','password'))
response.json()

Instead of entering the password in plain text, we can use authorization token
GitHub-->Settings-->Developer Settings-->Perssonal access tokens-->Generate new token
Give a name and select scope as user and generate token
response=requests.get(url,headers={'Authorization':'Bearer <token generated>'})
response.json()
